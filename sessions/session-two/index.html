Session Two — Python Programming

Learning objectives

By the end of this session, you will be able to

  • Create simple Python programs
  • Understand the role of variables
  • Create, edit, and run Python programs in CODIO.
  • Basic usage of the IDLE IDE
  • Understand how to implement sequence, selection, and iteration in Python
  • Understand the role of functions in programming
