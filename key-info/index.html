  • Module orientation

    Module name: Programming Primer: A Short Introduction to Computer Science

    Module code: BUCI057N0

    Module-level: 6

    Credit value: non-credit bearing module

    AIMS

    To provide an introduction to the content of the various introductory modules from our Computer Science and Data Science programmes.

    MODULE DESCRIPTION

    This "primer" course will act as a "jump-start" in your studies.

    The module is run in a blended style, where the cohort reads materials before class. Then, most of the face-to-face sessions are devoted to exercises, questions, and clarification of the materials. Apart from the first session, each subsequent session includes homework that consolidates the materials covered and prepares the student for the next session.

    The programming section of the module uses the Python programming language.

    Please note that this module requires an intensive study period and considerable effort outside of the scheduled sessions.

    MODULE LEARNING OUTCOMES

    By the end of this module, you will:

    • Understand the architecture of a computer.
    • Perform simple calculations using binary, octal, and hex-based arithmetic.
    • Understand the execution steps of a computer program.
    • Write small programs using the Python programming language.

    MATERIALS

    Materials associated with each session that comprises the module are listed in their respective sections. They are indicative of the content, and students should follow up on further materials. The brief notes from the various face-to-face sessions will also be posted on this site, and they will be uploaded following each session. Homework for each session can be found under the appropriate session section.

  • Provisional outline syllabus

    The syllabus is provisional and is adapted dynamically to cope with the varying requirements of the student cohort.

    • Introduction, overview, and orientation, including CODIO. Describing a Numerical Data Set
    • edit-compile-run; types, variables & assignment
    • The building blocks of programs
    • A refresher on basic arithmetic and simple set operations
    • Binary representation, Integer representations.
    • Importing and Describing Mixed Data Sets (pandas, matplotlib). Statistical Tests to Determine if Populations are Different
    • Functions (methods)
    • Number Systems. Machine execution cycle. Statistical Tests to Describe Relationships. Data Analysis Lab
    • Further Python programming. Creating Comparison and Composition Charts
    • Abstract Data Types. User-defined data types. Creating Distribution Charts. Creating Specialised Visualisations
    • An introduction to operating systems. Communicating Data Using Jupyter Notebook. Visualising Data and Communicating Results with Jupyter
    • Consolidation and review
  • Preparation for a class

    Preparation in advance of each class is essential to effective participation. Module guides will specify the basic materials required for each class. Students are expected to spend several hours preparing for each class, reading and taking notes from the materials. Failure to prepare for and participate in class is a disservice to your study goals and discourtesy to your fellow students, which can cause considerable frustration.
